SDA-PVTDet

论文阅读：SDA-PVTDet: A spatial-cross dual attention pyramid vision transformer detector for casting defect detection in radiography images

核心内容

本文提出SDA-PVTDet模型，包含三个核心模块：

  1. 空间交叉重叠嵌入模块（SO-Emb）：通过高度偏差卷积（HDConv）和宽度偏差卷积（WDConv）分别在垂直和水平方向创建重叠区域，结合深度可分离卷积和通道注意力机制，捕获模糊边界的局部连续性特征，减少下采样过程中的信息损失。
  2. 双注意力块：由差异窗口级联多头注意力（DWC-MSA）和全局感知多头注意力（GA-MSA）组成。DWC-MSA利用中心/对角/交叉差分操作提取局部梯度信息，GA-MSA通过膨胀卷积生成多尺度全局查询，结合维度压缩操作建立长程依赖，增强低可见度缺陷的特征表达。
  3. 特征回流金字塔网络（FBPN）：设计集中特征聚合模块（CFA），通过并行全局MLP分支和多分支局部卷积融合上下文信息，结合特征回流机制促进跨尺度特征交互，使用CBAM模块优化注意力权重，提升对不规则分布缺陷的检测能力。

SO-Emb

高度与宽度偏差卷积的数学表达

  • HDConv：沿垂直方向创建重叠区域，捕获高度维度连续性

    • 输入特征图 ，通过非对称卷积核（如7×4）提取垂直方向交互特征
    • 卷积步长（Stride）设为4，高度方向填充（Padding）为（3,0），保留边界信息
    • 输出

  • WDConv：沿水平方向创建重叠区域，增强宽度维度连续性

    • 使用非对称卷积核（如4×7），水平方向步长4，填充（0,3）
    • 输出

空间交叉特征融合

通过 跳跃连接(Skip Connection) 与 通道拼接(Concat) 融合双分支特征：

  1. 跨维度拼接：将  与  沿通道维度拼接

  2. 残差特征增强：对双分支求和结果进行深度可分离卷积

  3. 通道注意力加权：通过ECA模块动态调整通道权重

  4. 最终输出：加权拼接特征与残差特征相加

  • 非对称卷积核设计：通过高度>宽度（如7×4）或宽度>高度（4×7）的核尺寸，强化特定维度的局部特征提取能力
  • 交叉填充策略：垂直方向填充（3,0）与水平方向填充（0,3），确保特征图尺寸对齐且边界信息不丢失
  • 计算效率优化：
    • 深度可分离卷积（DWConv + PWConv）将计算量从
    • ECA模块仅引入  参数（r为压缩率），避免全连接层的高计算开销

双注意力模块

双注意力模块是SDA-PVTDet模型的核心创新之一，旨在解决低可见度射线图像中缺陷特征的局部梯度变化感知不足和全局语义关联弱的问题。该模块通过差异窗口级联多头注意力（DWC-MSA）与全局感知多头注意力（GA-MSA）的协同设计，实现了对缺陷特征的多层次建模。

差异窗口级联多头注意力（DWC-MSA）

设计动机

在低可见度图像中，缺陷的细微梯度变化往往被噪声掩盖。传统自注意力机制缺乏对局部梯度信息的特异性建模，导致关键特征丢失。DWC-MSA通过引入差分操作增强局部梯度感知能力。

全局感知多头注意力（GA-MSA）

设计动机

全局依赖建模对随机分布的大尺寸缺陷（如夹渣SI）至关重要。传统全局注意力计算复杂度高，且易受背景干扰。GA-MSA通过多尺度查询生成和维度压缩优化计算效率。

  • 全局特征增强将Q、K、V拼接后通过 全局压缩（GS） 生成紧凑特征 ，经PWConv和非线性激活后与逐元素相乘，增强通道-位置关联。

效果

GA-MSA的参数量仅为传统全局注意力的31%，但对大尺寸气孔（GC）的检测mAPL提升至97.6%，且背景误检率降低18%。

协同工作机制

双注意力模块通过级联结构整合局部与全局信息：

  1. 前向传播路径：DWC-MSA优先提取局部梯度特征，GA-MSA在此基础上建立长程依赖。
  2. 残差连接：每个子模块的输出通过残差连接保留原始特征，避免信息丢失。
  3. 跨尺度交互：DWC-MSA的窗口划分（7×7）与GA-MSA的膨胀卷积（最大dilation=4）形成互补，覆盖从像素级到区域级的特征层次。

实验验证

消融实验表明，双注意力模块使模型在ATBCD-DET数据集上的mAP50提升6.3%（从84.9%至91.2%），尤其对低对比度缺陷（如伪缺陷PD）的召回率提高12.7%。

特征回流金字塔网络（FBPN）

设计背景与核心问题

传统特征金字塔网络（FPN）通过自上而下的路径融合多尺度特征，但在铸造缺陷检测中存在两个关键问题：

  1. 语义信息割裂：深层特征（高层语义）与浅层特征（细节信息）融合时，跨层级语义关联不足，难以捕捉小缺陷的细节特征。
  2. 单向信息流限制：仅通过自上而下的特征传递，缺乏浅层到深层的反馈机制，导致不规则分布缺陷的上下文建模能力受限。

FBPN核心结构

FBPN由集中特征聚合模块（CFA）和特征回流机制组成，具体结构如下：

集中特征聚合模块（Centralized Feature Aggregation, CFA）

功能：增强多尺度特征的语义表达，融合全局上下文与局部细节。

结构：

  • 全局信息分支：

    • 深度可分离卷积：采用3×3深度卷积提取跨通道特征，降低计算量。
    • 多层感知机（MLP）：通过两层全连接（通道数扩展比为4）建模全局依赖关系。
    • 残差连接：保留原始特征信息，避免梯度消失。

  • 局部信息分支：

    • 多分支卷积结构：
      • 路径1：1×1卷积降维 → 3×3卷积捕捉局部纹理。
      • 路径2：1×3+3×1非对称卷积 → 空洞卷积（dilation=5）扩大感受野。
      • 路径3：3×1+1×3非对称卷积 → 空洞卷积（dilation=5）提取长条形缺陷特征。
    • 通道拼接与压缩：将多分支输出拼接后通过1×1卷积压缩至原通道数。

  • 特征融合：

技术优势

  1. 双向信息流：
    • 通过特征回流机制实现浅层→深层（细节补充）和深层→浅层（语义增强）的双向交互，提升对微小缺陷（如气孔）和大型缺陷（如裂纹）的联合检测能力。
  2. 自适应感受野：
    • CFA模块中空洞卷积与非对称卷积结合，可动态适应不同形态缺陷（点状气孔 vs. 线状冷隔）。
  3. 计算效率优化：
    • 深度可分离卷积减少75%参数量，CBAM注意力仅增加0.03M参数，在RTX 3090上推理速度达32 FPS。

实验验证

在ATBCD-DET数据集上的消融实验表明：

  • CFA模块贡献：mAP50提升4.2%（从84.9%→89.1%），小缺陷检测精度（mAPs）提升7.6%。
  • 回流机制效果：在YOLOv5-S模型中引入FBPN，mAP50从80.7%提升至82.9%，伪缺陷（PD）召回率提高6.3%。
